# Nifty Algo Trader

A Python-based algorithmic trading platform for Angel One SmartAPI.

## Objectives

- Connect to Angel One SmartAPI
- Fetch live market data
- Analyze option chains
- Implement trading strategies
- Backtest strategies
- Paper trading
- Fully automated live trading
- AI-powered trade analytics

## Tech Stack

- Python
- Git & GitHub
- SmartAPI
- Pandas
- NumPy
